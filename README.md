# 🚀 SkillGap AI

SkillGap AI is a full-stack web application that analyzes resumes and identifies skill gaps based on user-defined career goals. It provides personalized learning roadmaps using a combination of Node.js and Python.

---

## ✨ Features

* 📄 Resume upload (PDF, DOCX, TXT)
* 🔍 Skill extraction & gap analysis
* 🎯 Career goal–based recommendations
* ⚡ Fully offline Python analysis engine
* 🎨 Modern responsive UI
* 🔐 Optional user authentication & history tracking

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Python Engine:** Resume parsing & analysis
* **Database:** SQLite3
* **File Handling:** Multer

---

## ⚙️ How It Works

1. Upload your resume
2. System extracts skills using Python scripts
3. Enter your target role
4. Get skill gap analysis + learning roadmap
